# 📚 ACALED – Book Lending Service

Welcome to **ACALED**, a modern and user-friendly **Book Lending Service** designed to make reading more accessible. Browse, borrow, purchase, and return books through a simple and efficient platform built with a clean and minimalist interface.

---

## 📖 About ACALED

ACALED is a web-based book lending system that allows users to conveniently manage their reading experience. Whether you want to borrow your next favorite book or purchase one to keep, ACALED provides a seamless and hassle-free process.

---

## ✨ Features

- 📚 Browse available books
- 📖 Borrow books with ease
- 💳 Purchase books online
- 🔄 Return borrowed books
- 🎁 Earn reward points from every purchase
- 👤 User-friendly dashboard
- 📱 Responsive and minimalist interface

---

## 💡 Why ACALED?

We believe that reading should be simple and accessible. ACALED was built to provide an efficient book lending experience while encouraging users to continue reading through a rewarding point system.

- Simple and intuitive interface
- Fast borrowing and returning process
- Reward points on every purchase
- Clean and modern design
- Easy navigation for all users

---

## 🎁 Reward System

Every purchase earns you **reward points** that can be used toward future book borrowing or purchases. These rewards help create a more engaging experience while supporting the continuous improvement of the platform.
